⊕ BASE-19 SYMBOL DICTIONARY — BRACKETED SINGLE-DIGIT EXTENSION
Values 0–9 use standard digits. Values 10–18 are written as bracketed single digits — [10] through [18] — each one a single B19 place-value, never split into separate decimal digits.
Critical rollovers:   19dec=10  ·  38dec=20  ·  70dec=3[13]  ·  291dec=15[6]  ·  361dec=100
